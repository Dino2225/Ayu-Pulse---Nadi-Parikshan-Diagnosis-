# Publishing the project to GitHub

## Recommended commands

```bash
git init
git add .
git commit -m "Initial Nadi Parikshan prototype"
git branch -M main
git remote add origin <YOUR_GITHUB_REPO_URL>
git push -u origin main
```

If you already have a repository:

```bash
git add .
git commit -m "Update README, docs, and model loading"
git push
```

## Suggested workflow

1. Verify the app runs locally.
2. Add a `demo.gif` or screenshot if available.
3. Make one clean commit for the initial upload.
4. Push to `main` or a named branch.

## Optional follow-up

- Create a GitHub release with the README and demo screenshot.
- Add a short `project-description.md` or `data-card.md` for portfolio context.
- Use the issue template to track accuracy improvements.
